# 聖誕夜的謎 (The Mystery of Christmas Eve)

這是一個基於 HTML/CSS/JavaScript 的互動式文字冒險（視覺小說）遊戲，由 [jellyfishyjammy] 開發。玩家將扮演一名偵探，在時間停止的聖誕村裡，找出造成一切的罪魁禍首。

## 🎮 遊戲玩法

* **場景探索：** 點擊畫面上的「熱點」圖示（例如 🚀 實驗室、✨ 檢查基座）以發現線索。
* **對話與證據：** 在對話場景中，可以對嫌疑人進行質問。
* **指認：** 掌握足夠證據後，在中央廣場（Hub）選擇指認嫌疑人。

## 📜 故事背景

聖誕夜 11:59，時間就此凝固。你必須在時間再次流動前，從三位沒有被凍結的嫌疑人中，找出真正的犯人。

## 🔒 隱私權聲明與資料收集

本遊戲透過 **Google Analytics 4 (GA4)** 收集**匿名**的遊玩數據，以幫助開發者優化遊戲難度和使用者體驗。

**我們收集的數據是完全匿名的，且不包含任何個人識別資訊 (PII)。** 收集的數據包括：

* **遊玩行為：** 玩家在遊戲中點擊的按鈕、選擇的對話選項、點擊熱點的次數。
* **錯誤次數：** 嘗試指認錯誤嫌疑人的次數。
* **技術資訊：** 瀏覽器類型、裝置類型、大概地理位置（國家/地區級）。
* **目的：** 分析遊戲流程中的卡關點、故事分支的平衡性，以及玩家平均遊玩時長。

如果您不希望被追蹤，您可以在瀏覽器中啟用「請勿追蹤 (Do Not Track, DNT)」功能，或使用隱私瀏覽模式進行遊戲。
